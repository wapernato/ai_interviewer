# AI Interviewer

AI Interviewer — backend-приложение для тренировки технических собеседований.  
Пользователь выбирает тему, активный AI-профиль задаёт вопрос, пользователь отправляет ответ, после чего система сохраняет вопрос и ответ в PostgreSQL и возвращает предварительный feedback.

Проект разрабатывается как учебный backend-проект для ознакомительной практики.

---

## Цель проекта

Цель проекта — создать личного AI-интервьюера, который помогает пользователю готовиться к техническим собеседованиям.

Основной сценарий работы:

1. Пользователь выбирает тему.
2. Система берёт активный AI-профиль.
3. AI-профиль генерирует вопрос по выбранной теме.
4. Пользователь отправляет ответ.
5. Ответ сохраняется в базе данных.
6. Система возвращает feedback.
7. Пользователь может посмотреть историю вопросов и ответов.
